EDSR
====

Source:
- Paper: B. Lim, S. Son, H. Kim, S. Nah, K. M. Lee, "Enhanced Deep Residual
  Networks for Single Image Super-Resolution", CVPRW 2017.
- Code and pretrained weights: https://github.com/sanghyun-son/EDSR-PyTorch
- Pretrained baseline weights:
  - edsr_baseline_x2-1bc95232.pt
  - edsr_baseline_x3-abf2a44e.pt
  - edsr_baseline_x4-6b446fab.pt

The converted ONNX models are derived from the official MIT-licensed EDSR
baseline pretrained weights. The conversion script wraps the network so QSVEnc /
NVEnc / VCEEnc --vpp-onnx can use the models as 3-channel RGB super-resolution
models with [0,1] input and output.
